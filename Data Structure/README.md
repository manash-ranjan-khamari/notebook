## Big-O Notation
- Complexity in ascending order can be:

| Description       | Big-O     |
| ----------------- | --------- |
| Constant time     | O(1)      |
| Logarithmic time  | O(log(n)) |
| Linear time       | O(n)      |
| Linearithmic time | O(nlog(n))|
| Quadratic time    | O(n^2)    |
| Cubic time        | O(n^3)    |
| Exponential time  | O(b^n)    | b > 1
| Factorial time    | O(n!)     |
